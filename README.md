# Wayfarer's Codex: Scrying Station

Turn Discord into a warm, translucent fantasy scrying desk. This BetterDiscord
theme layers a candlelit scholar's table behind clear, low-glow panels so chat,
channels, and calls feel like part of the same scene.

The artwork is original and fantasy-inspired. It does not use licensed settings,
characters, logos, maps, or tabletop game art.

## What You Get

- A lightweight, CSS-only BetterDiscord theme; no companion plugin is needed.
- An offline fantasy desk background embedded directly in the theme.
- Translucent panels throughout servers, Direct Messages, member lists, and
  activity views.
- Native Discord unread indicators and server role colors for member names.
- A clean, low-glow chat composer designed to stay comfortable while typing.
- Video and call surfaces that blend into the desk scene where Discord permits.

## Install

1. Download and extract the latest release zip.
2. Move `WayfarersCodex-ScryingStation.theme.css` into your BetterDiscord
   themes folder.
3. Reload Discord and enable **Wayfarer's Codex: Scrying Station** in
   BetterDiscord's Themes settings.
4. Keep BetterDiscord **Custom CSS** disabled if it contains an older Wayfarer
   setup, since that old styling can interfere with this theme.

The background is embedded in the CSS file, so the theme does not need an
internet connection to show the desk scene.

## Included Files

- `WayfarersCodex-ScryingStation.theme.css` - the theme to install.
- `preview-scrying-station.html` - an offline preview page.
- `assets/` - the scene artwork in PNG and WebP formats.

## Adjust Transparency

The panels are set to 10% opacity by default. To make them darker or clearer,
edit this value near the top of the theme:

```css
--wc-glass: rgba(30, 19, 13, 0.10);
```

Increase `0.10` for darker panels; decrease it for more of the desk scene.
